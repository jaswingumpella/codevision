# Dashboard Review Skill

When reviewing frontend graph/ and usecases/ components:

1. Sigma.js canvas initializes without errors
2. Semantic zoom transitions are smooth (package -> class -> method)
3. Search bar returns fuzzy-matched results within 200ms
4. Filter panel correctly hides/shows nodes by type
5. Detail panel shows complete metadata for clicked node
6. Context menu offers relevant actions (show deps, impact analysis, etc.)
7. Legend colors match actual node rendering
8. Loading states shown during async operations
9. Error boundaries prevent crashes from propagating
10. Components are lazy-loaded for performance
